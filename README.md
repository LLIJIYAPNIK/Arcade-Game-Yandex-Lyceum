# Описание работы над проектом
---
База была взята из официальной документации, в коде можно увидеть много комментариев из примеров Arcade.

Применённые технологии:

+ Система выносливости и здоровья
+ Анимация главного героя и спрайтов на карте
+ Коллизии
+ Система частиц
+ Параллакс
+ Пиксель-арт (7 часов жизни ради pixelated=True)

Проект будет дорабатываться летом.
